# htmltool

HTML pages with some functions:
- json verify and format.
- timestamp convert.
- regex.
- base64 and url encode/decode.
- password generate.

### usage:

- Replace vue and element-plus links in `index.html` to cdn links.
- Or install vue and element-plus package in the same directory of `index.html`.

Then open `index.html` in broswer.
